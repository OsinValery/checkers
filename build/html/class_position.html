возвращает все ходы фигуры на поле (x,y) не используйте line и free вне класса

Parameters
x- номер вертикали, на которой стоит фигура 0..7
y- номер горизонтали 0..7
free- степень свободы во время движения если установить false, фигура может искать только ходы, где она рубит врага
line- это специальный параметр для дамки когда она несвободна, она может лишь двигаться только по 1 диагонали, проверяя все перпендикулярные линии на наличие врага для новой реккурсии

line задаёт номер диагонали, откуда движется фигура

0 - ничего

1 - вверх направо

2,3.4 - диагонали по часовой стрелке

4 - фигура движется из верх-лево в низ-право
